# 🔐 cyper.js
Encrypte and Decrypte in CLI!!
## 📝 Features
SHA-1, SHA-256, SHA-512, AES-128, AES-256
## 📥 Installation
```
npm install -g @joing/cyperjs
```
## ⌨️ Usage 
Just Type "cyper"!!
```
Usage: cyper [option]

Options:
  -v, --version  output the version number
  -h, --help     display help for command
```
